# PS2-Nano-Remote
Nano remote control to On / Off / Reset PS2 ( from SCPH-5000x to SCPH-9000x )

<h3 align="left">You will need:</h3> 

  
- RP2040 Zero  
- IR Diode  
- CR2032 Coin Battery  
- Tact Button
